## Installation

**Note**: After installation, the widget may not appear immediately. Please attempt 1-2 queues - it takes a moment to load on first use.

Install from the RemNote Plugin Store, or build from source at [github.com/moritzWa/queue-productivity-tracker](https://github.com/moritzWa/queue-productivity-tracker)

## Usage

The plugin automatically activates when you enter a flashcard queue, displaying:

- **Clock**: Current time
- **Session**: Study session duration
- **Speed**: Cards/minute (last 5 minutes only)
- **Status**: Cards completed and accuracy
- **Expected**: Time remaining and completion ETA
